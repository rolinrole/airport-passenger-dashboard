# Airport Passenger Dashboard

A simple interactive airport passenger dashboard built with HTML, CSS, and JavaScript.

## Project Overview

This project displays airport flight information and allows users to search flights by flight number, airline, gate, destination, or status.

## Features

- Flight information table
- Search functionality
- Dynamic JavaScript filtering
- Airline style dashboard layout
- Responsive page structure
- Status based visual styling

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Visual Studio Code
- Git
- GitHub
